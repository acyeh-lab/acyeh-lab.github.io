Spillover risk

Pairs where emission peaks differ by < 25 nm and share a laser are flagged automatically. Add free-form notes on each pair — they're saved with the panel and pushed to GitHub on Save.

Saved Panels

Panels are stored in your browser and — if a GitHub token is set — automatically pushed to flow-panels-data.json in the site repo on every Save. Use the manual Sync button to force a push.
